# 📈 Linear Regression - Placement Prediction

## 📌 Project Overview
This project demonstrates how to build a Machine Learning model using **Linear Regression** with **Scikit-learn** to predict student placement packages based on their CGPA.

## 🎯 Objective
Predict the expected placement package of a student using their CGPA.

## 📂 Dataset
- Placement Dataset
- Input Feature: CGPA
- Target Variable: Package

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
